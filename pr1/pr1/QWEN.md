# Реализация: Работа с графикой в C# (Windows Forms)

## ✅ Реализованный функционал

### Архитектура

Приложение построено по принципу ООП с использованием наследования:

```
Shape (абстрактный базовый класс)
├── Line (линия)
├── Ellipse (эллипс)
└── Polygon (многоугольник)
```

Каждая фигура реализует методы:
- `Draw(Graphics g)` — рисование фигуры
- `Erase(Graphics g)` — стирание фигуры (рисование цветом фона)

## 🎨 Интерфейс приложения

### Элементы управления

**GroupBox "Управление"** (слева, ширина 380px):
- `ComboBox` — выбор фигуры (Линия, Эллипс, Многоугольник)

**GroupBox "Сообщения"** (снизу):
- `Label` — вывод статуса операций

## 🔧 Форматы ввода данных

### 📏 Линия
```
x1,y1,x2,y2
цвет
толщина

Пример:
0,0,100,100
red
2
```

### ⚪ Эллипс
```
x,y,radX,radY
цвет
толщина
текст

Пример:
100,100,50,30
blue
2
Текст
```

### 🔷 Правильный многоугольник
```
x,y,стороны,радиус
цвет
толщина

Пример:
150,150,6,50
green
3
```

### 🔺 Произвольный многоугольник
```
x1,y1,x2,y2,...
цвет
толщина
[заливка]

Пример:
50,50,100,50,100,100
red
2
```

**Поддерживаемые цвета:** red, green, blue, yellow, cyan, magenta, white, black, orange, purple, pink, gray

## 🧠 Логика работы

### Хранение фигур
```csharp
private readonly List<Shape> shapes = new();
```
Все нарисованные фигуры сохраняются в списке для последующего стирания.

### Инициализация графики
```csharp
g = panelCanvas.CreateGraphics();
ClearCanvas(); // очистка области
buttonEnable.Enabled = false;
```

### Рисование
1. Создание фигуры через `CreateShapeFromParams()`
2. Установка `BackgroundColor = panelCanvas.BackColor`
3. Вызов `shape.Draw(g)`
4. Добавление в список `shapes`

### Стирание
1. Берётся последняя фигура из списка
2. Вызов `shape.Erase(g)`
3. Удаление из списка

## 🐛 Исправленные баги

### 1. Парсинг точек произвольного многоугольника
**Проблема:** Кнопка "Доб." добавляла только первую точку  
**Решение:** Поддержка формата `x1,y1,x2,y2,...` — все точки добавляются за один раз

### 2. Парсинг цвета и толщины
**Проблема:** Требуется пустая строка в начале для правильного парсинга  
**Решение:** Для произвольного многоугольника параметры парсятся отдельно из `textBoxParams` после точек

### 3. Перекрытие подсказок
**Проблема:** `ListBox` с точками перекрывал текст подсказок  
**Решение:**
- Увеличена ширина формы с 919 до 1050px
- Увеличена ширина `groupBoxControls` с 280 до 380px
- `ListBox` и кнопки перемещены вправо
- `labelParams` расширен для отображения полных подсказок
