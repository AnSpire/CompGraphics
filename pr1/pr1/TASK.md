# Начальное задание: Работа с графикой в C# (Windows Forms)

## Цель

Нужно разработать приложение на C# (Windows Forms), которое умеет:

* рисовать простые фигуры (линия, эллипс, многоугольник);
* стирать отдельные объекты;
* очищать всю область рисования;
* управлять параметрами фигур через интерфейс;
* реализовать фигуры через классы с методами `Draw()` и `Erase()`.

---

## 🧠 Роль модели

Ты выступаешь как опытный разработчик C# (WinForms + System.Drawing), который:

* пишет чистый и расширяемый код;
* использует ООП (наследование, абстракции);
* минимизирует зависимость логики от UI;
* при необходимости предлагает улучшения.

## 🎨 Поддерживаемые фигуры

---

### 📏 1. Линия

#### Свойства:

* начальная точка (`Point`)
* конечная точка (`Point`)
* толщина
* цвет линии
* цвет фона (для стирания)
* стиль линии (`DashStyle`)
* начало/конец (`StartCap`, `EndCap`)

#### Поведение:

* Рисование через `Pen`
* Стирание = повторное рисование цветом фона

---

### ⚪ 2. Эллипс

#### Свойства:

* центр (`Point`)
* радиус по X
* радиус по Y
* цвет контура
* цвет заливки
* цвет фона
* текст (опционально)

#### Особенности:

* `DrawEllipse` — контур
* `FillEllipse` — заливка
* `DrawString` — подпись

Координаты:

```csharp
x = center.X - radiusX;
y = center.Y - radiusY;
```

---

### 🔷 3. Многоугольник

#### Варианты:

1. Произвольный (по точкам)
2. Правильный (n-угольник)

---

#### Свойства:

* массив точек (`PointF[]`)
  или:
* центр
* количество сторон (`n`)

Также:

* цвет контура
* цвет заливки
* цвет фона
* толщина линии

---

### 🔺 Произвольный многоугольник

Добавление точки:

```csharp
int x = Convert.ToInt16(textBoxX.Text);
int y = Convert.ToInt16(textBoxY.Text);
listBox1.Items.Add(x + " " + y);
```

Парсинг:

```csharp
string s = listBox1.Items[i].ToString();
string[] a = s.Split(' ');
points[i] = new PointF(
    Convert.ToInt16(a[0]),
    Convert.ToInt16(a[1])
);
```

Рисование:

```csharp
g.DrawPolygon(pen, points);
g.FillPolygon(brush, points);
```

---

### 🔷 Правильный многоугольник

Формула:

```csharp
double angle = 2 * Math.PI / n;
```

Генерация точек:

```csharp
for (int i = 0; i < n; i++)
{
    float x = center.X + R * (float)Math.Cos(angle * i);
    float y = center.Y + R * (float)Math.Sin(angle * i);
}
```

## 🧩 Требования к архитектуре

* Использовать ООП
* Каждая фигура — отдельный класс
* Обязательные методы:

  * `Draw(Graphics g)`
  * `Erase(Graphics g)`
* Не дублировать код
* Минимизировать зависимость от UI

## 🧠 Рекомендуемая архитектура

Базовый класс:

```csharp
abstract class Shape
{
    public abstract void Draw(Graphics g);
    public abstract void Erase(Graphics g);
}
```

Наследники:

* Line
* Ellipse
* Polygon
