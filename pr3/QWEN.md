Разработай приложение на C# с использованием Windows Forms, в котором реализовано построение графиков функций с помощью компонента Chart.

### Требования:

1. **Интерфейс:**

   * Размести на форме компонент `Chart`.
   * Добавь элементы ввода (например, TextBox) для ввода **двух или более математических функций** (например: `sin(x)`, `x^2`, `cos(x)` и т.д.).
   * Добавь кнопку для запуска построения графиков.

2. **Парсинг выражений:**

   * Реализуй преобразование введённых строк в вычисляемые функции.
   * Используй библиотеку **NCalc** (подключить через NuGet) для вычисления значений функций.

3. **Построение графиков:**

   * Построй графики всех введённых функций на одном Chart.
   * Для каждой функции используй отдельную серию (`Series`).
   * Задай разные цвета линий для разных функций.

4. **Настройка графика:**

   * Подпиши оси (X и Y).
   * Настрой диапазон значений по оси X (например, от -10 до 10).
   * Добавь сетку (Grid), если необходимо.

5. **Маркеры:**

   * Для точек графика включи маркеры.
   * Используй разные `MarkerStyle` (круги, квадраты, треугольники и т.д.) для разных функций.

6. **Дополнительно:**

   * Позволь менять тип графика (линейный, точечный и т.д.).
   * Обработай ошибки ввода (например, некорректные формулы).
   * Обеспечь плавную и понятную визуализацию.

7. **Результат:**

   * Приложение должно отображать несколько графиков одновременно.
   * Код должен быть структурированным и понятным.
